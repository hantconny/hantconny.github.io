凌宇凡，12月12日出生于清水，这一天与当年双十二事件发生的时间重合。

易筱凡，12月12日出生于清水，这一天与当年双十二事件发生的时间重合。

他们出生在一个产房，睡在相邻的小床，有一样晶亮的瞳仁。

<br/>
<br/>

“你叫什么呀？”

“凌宇凡。”

“好好听的名字啊，我叫易筱凡。”

“你的名字也很好听。”

<br/>
<br/>

“凌宇凡，我们来扮家家，你当爸爸，我当妈妈。”

“恩，谁当宝贝儿呢？”

“她好了。”易筱凡指着手里的芭比。

“恩。”

“爸爸，起床了。”凌宇凡把眼睛睁开，“妈妈要做饭给爸爸和宝贝儿吃。”

“恩。”

“爸爸去上班，妈妈做家务。”

“忽忽，爸爸下班了。爸爸辛苦了。妈妈帮爸爸拿衣服和包。”

“恩。”凌宇凡很用力的点头。

“妈妈做饭给爸爸和宝贝儿吃。”

“恩。”

<br/>
<br/>

“凌宇凡，你妈妈叫你名字吗？”

凌宇凡抬起头，是晶亮而疑惑的瞳仁。

“叫宝宝？”

“恩。”凌宇凡很用力的点点头。

“你就会恩。”易筱凡说着也学他很用力的点头，“我妈妈也这么叫我。”一抹甜美的笑容，透过早春的阳光，自下而上的散发，弥漫到整个初春的空气里。

<br/>
<br/>

“你看我的红裙子好看吗？”

“恩，好看。”

“妈妈给我的生日礼物”说着很自由的，自由的转了个圈，红色的裙摆散开，像血色的昙花一样耀目绽放，“你妈妈给你什么做生日礼物啊？”

凌宇凡伸出手，手腕有一块与小手不相称的swatch。

“我以后做你的新娘好不好？”

“恩。”

“你就会恩。”

“恩。”

<br/>
<br/>

“易筱凡，”凌宇凡的脸在冬日的寒冷里红彤彤的，“我要搬家了。”

“会搬去很远的地方去。”

“不知道。”

“你会经常来看我吗？”

“你说过要做我的新娘的。”

“恩。”

“你就会恩。”

斜斜长长的影子串联着覆盖在水泥地上。

<br/>
<br/>

一个礼拜后，凌宇凡搬到了隔壁镇上。那个镇叫永青，那年他5岁。

易筱凡5岁那年，妈妈从她的生活中消失。1个月以后，她的爸爸带着她和另外一个女人离开了清水，去向不明。

<br/>
<br/>

“你个要死的贱丫头，衣服还不洗，你想死啊。”

“诶呀，真是，老凌啊，隔壁的那家真是……”凌妈妈在厨房嚷嚷着。

“恩，我在考虑是不是该搬家了，环境对宇凡的成长也不好。”

“是啊，”凌妈妈压低声音，“儿子马上也要考大学了。”

“你个死丫头，B养的……”

“好了，注意点！”凌妈妈打开门，冲对面吼了句。然后又低低的咕噜了一句，“怎么也是你女儿，没见过这样的家长。”

“怎么啊，我教训我女儿关你什么事啊？可怜她，你领回去养啊！”

“哎呀！”凌妈妈被对面女人突如其来的架势吓了一跳，赶紧关了门，还不停顺着自己的胸口，“没见过这样的人，没见过。诶，快回去做你的作业啊。”凌妈妈瞥见房间里探出头的儿子，立即停了手上的动作。

晚上的雨很大，凌宇凡躺在床上，仿佛听到一群群的候鸟迁徙的声音。

早上的雪不大，风微凉，凌宇凡光着身子套上羊毛衫，他喜欢那种隐约的刺痛的感觉，也喜欢伴随体温渐渐蒸发出的毛线味道。

易筱凡在前面走，穿同样的校服。

凌宇凡跟在她后面，没有围巾，他不喜欢脖子被羊毛刺痛的感觉，那样他觉得很不自然，因此他总是穿圆领的羊毛衫，上面有暖色调的横条。

“喂。”

“恩。”

“你叫什么名字？”

“易筱凡。”

“哦，我叫凌宇凡”

高三的楼下是第一次模拟考试的成绩。第一名是易筱凡，第二百名是凌宇凡。

“苏三…离了…洪桐县…”凌宇凡努力集中精力听着英文，但是隔壁女人唱的的确很吸引人。

“妈，我要听英语，你不要唱了。”

“将身赖在…大街前…未”

“哐…”是盘子或者什么陶瓷破裂的声音。

“你个B养的，盘子不要钱啊？”

“打不死你个B养的东西。”

“诶，老凌啊，快点想想办法啊……”

“我已经在找房子了……”

“Today , I have a dream that ……”凌宇凡调高了音量，耳朵里的英语突然又清晰起来了。

高三的楼下是第三次模拟考试的成绩。第一名是凌宇凡，第二百名是易筱凡。

学校的门口围了很多人，警察已经拉起了警戒线。

“听说一个高三的女生自杀了呀……”

“大概考试的压力太大了吧……”

“现在的学生啊，真是作孽哦……”

“想我们那个时候，光荣交白卷啊……”

“那是什么时候的事了哦……”

易筱凡走在前面，凌宇凡追上去，握住她的手，很用力，很用力，直到易筱凡把手抽走，晶亮的眸子注视着他，然后突突突的消失在宛如覆盖着铜浆的大理石地面上。

凌宇凡倚在G大的玉兰树下，一张明信片和一张照片从手中的书页落下，旋转如飘落的玉兰花瓣。

照片上是监狱里的易筱凡。
