在凌宇凡没有遇见杜菁菁的时候，他一直失眠。早早上床，辗转，难以入睡。凌晨醒来，泪流满面。

12月12的凌晨，凌宇凡再次泪流满面的醒来。

他点进所有订阅的好友博客，在这个失眠的午夜，他迫切想知道他们的近况。

猫的空间里，背景音乐并没有立即出现。而它一旦出现，带给凌宇凡的是意想不到的平静。

他惊奇于那样的音乐，很轻柔的糅进人的精神深处，像藤蔓一样爬满思维的墙壁，像熨斗一样，以最温柔的方式抚平褶皱。

他斜倚在椅子上。那一夜，他前所未有的平静，没有噩梦，没有眼泪。

猫的空间隐藏了播放器，他不知道那首音乐的名字。他也不会去寻找，只是失眠的时候会打开猫的空间，一直听，一直听，直到第二天有毛茸茸的太阳。

那首歌能告诉我名字么？凌宇凡留言。

Promise。

邮箱里出现那首歌的时候，他犹豫着该不该下载。你几乎不可能找到。几乎是在他按下确定的时候，11个文件传送界面一起出现。他们有不同的名字，不同的季节，不同的心情。

“我一直失眠，但是那个夜里，我睡着了。”

“恩。”

“你为什么叫黒い翼の貓呢？”

“我喜欢猫。”

“那些日文代表什么呢？”

“长着黑色翅膀的猫吧，我猜。”

“就这么理解吧。”

他们开始谈论学校，音乐，电影，猫。以及他们相遇的经历。那成了凌宇凡生活的一部分。

浅吟低唱的女声，配上淡淡的钢琴音，自然而真实。

“以后就不会再失眠了。”

“为什么都和猫有关呢？”

“因为他们一直孤独，却一直不觉得孤独，他们属于黑暗，却一直可以洞见黑暗。他们很勇敢。”

<br/>
<br/>

“你养猫吗？”

“养，但是只是断断续续的养。”

“现在有猫在你身边吗？”

“送人了。”

“舍得？”

“它不乖。而且，就我喜欢猫罢了。”

<br/>
<br/>

凌宇凡开始按时睡觉起床。冬天光着身子穿有暖色调横条的羊毛衫，他喜欢刺痛的感觉。夏天穿纯厚的白色衣服，棉的，麻的，混纺的，但都是纯厚的，因为不容易有缱绻的痕迹。杜菁菁依旧像她自己描述的那样，穿黑色的衣服，像一只猫一样生活，行走。她从不装饰自己，素面，随意收拾自己的头发。但依旧美丽。

只是有一天，他们的关系突然间有了飞跃。

“大爷我快疼死了。”

“你怎么了？凌不习惯杜菁菁变成一个粗俗的女人。”

“我神经疼。”

“遗传的吗？”

“我恨那个女人。”

“你妈妈吗？”

“她不想我来到这个世界，但我还是来了，只是拜她所赐了一身的病。”

没有一句是回答。

凌宇凡开始知道杜菁菁一直去就近的医院理疗，为了医治神经疼痛。
杜菁菁开始知道凌宇凡一直去就近的网络公司兼职，为了逃避无聊的课。

“你以前没有告诉我你有神经疼。”
“以前和你不熟。”
“那么现在呢？现在和我很熟吗？”

他没有问她要照片，他不知道她长的什么模样。高矮胖瘦，只是一般人心中的人类形象，她在他的心中，是像猫一样安静而勇敢的女子。
她从没有想象过他的模样，但是她知道有一天，她会在人群中一眼认出他来。因为他的气息。或者像莉莉说的，那是属于他的以太。
那种以太，没有烟草的气息，没有酒精的神韵。那种以太，有睡眠熨烫过的温柔的缱绻。

“你有男朋友吗？”

“没有。你呢？”

“我有。”

“我给你看她的照片。”

“好。”

“她很简单，像柠檬一样。”

“是的。”

<br/>
<br/>

“她爱你吗？”

“爱。”

“爱到什么程度。”

“我不知道。”

“那她不爱你。”

“为什么？”

<br/>
<br/>

“我们很暧昧。”

“是么？他会用问句。反客为主，这是生活的哲学。他没有压力。”

“暧昧又何尝不好。”

“暧昧是勇敢的游戏。”

“恋人的褪去形式是朋友，暧昧的褪去形式是陌生人。像你和我。”

“害怕的人才会用承诺来牵绊情感。”

<br/>
<br/>

凌宇凡从来没有想过，有一天会和杜菁菁见面。但她是个勇敢的女子。像猫一样。

“我在P市的客运西站，你能来接我吗？”
“好的。”凌发出短信的时候是在宿舍后面的江边。是巧合让他出现在那里。他没有留住那个柠檬样的女子。江面还有回旋的涡流。

杜菁菁一眼就看到了那个属于她的以太。那种以太，没有烟草的气息，没有酒精的神韵。他穿干净的纯厚的白色衣服。没有褶皱和缱绻。她一袭黑色，像猫一样行走生活。

他带她去酒店。她没有行李和戒指。她只是作一次单人旅行。她没有目的。她没有朋友在P市。他是她的一个不认识的朋友。她不认为他可以信任。但是她找到了他。
她要了间单人房。
他们接吻。但只接吻。

“她不爱你。”她紧紧的抱住他的躯体，他有强健的肌肉，她感到安全。她得到的回应只是沉默。“她真的不爱你。而我也不爱你。”

他们在网上相识。
